## Overview

The **Azoth Framework** is a modern and optimized C++ library that crystallizes advanced knowledge of Machine Learning, Deep Learning, and Autonomous Systems into reusable, high-performance components.

### Main Features

- **High Performance**: SIMD optimizations, cache-blocking, parallel processing
- **Modular**: Independent and composable components
- **Complete**: From basic tensors to autonomous agents
- **Production Ready**: Optimized and tested code
- **Documented**: Mathematical formulas and practical examples
- **Customizable**: Easy extension and integration
